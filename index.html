ตั้งค่า Firebase ก่อนใช้งาน
เพื่อให้ทุกคนในทีมเห็นข้อมูลชุดเดียวกัน ต้องเชื่อมต่อกับ Firebase ก่อน (ฟรี ใช้เวลาประมาณ 5 นาที)
1
ไปที่ console.firebase.google.com → สร้าง Project ใหม่ → ตั้งชื่อ เช่น team-trip-2026
2
ใน Project → คลิก Build → Realtime Database → Create Database → เลือก asia-southeast1 (สิงคโปร์) → Start in test mode
3
กลับหน้า Project Overview → คลิกไอคอน </> (Web) → ลงทะเบียน App → คัดลอก firebaseConfig ที่ได้
4
เปิดไฟล์ HTML นี้ใน Notepad/VSCode → หาข้อความ PASTE_YOUR_API_KEY_HERE → แทนที่ค่าทั้ง 7 บรรทัดด้วย config ที่คัดลอกมา → บันทึกไฟล์
5
โฮสต์ไฟล์ฟรีด้วย Netlify Drop → ลาก HTML ไฟล์ไปวาง → คัดลอก URL → แชร์ให้ทีม 🎉
⏱️ Test mode ให้อ่าน/เขียนได้ฟรี 30 วัน — เหมาะสำหรับโหวตทริป หากต้องการใช้นานกว่านั้นให้อัปเดต Rules ใน Firebase console
กำลังเชื่อมต่อ Firebase...
